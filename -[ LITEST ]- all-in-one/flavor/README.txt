本目录用于切换不同的皮肤样式，主要针对 std 模式。
This folder is used for switching the skin *flavor*, which is mostly about the
std mode.


==========================================================================
 目录结构    | 说明
 Directory   | Description
-------------|------------------------------------------------------------
 _Shield/    | _Shield 皮肤元件 / Skin elements from the '_Shield' skin
   APPLY.bat | 应用此皮肤的脚本 / Script to apply this flavor
-------------|------------------------------------------------------------
 prjct-X/    | prjct. X 皮肤元件 / Skin elements from the 'prjct. X' skin
   APPLY.bat | 应用此皮肤的脚本 / Script to apply this flavor
-------------|------------------------------------------------------------
 etc/        | 支持及公用文件 / Supporting and common files
==========================================================================


使用方法 / Usage
================
  [[重要提示 / IMPORTANT]]
  以下操作会覆盖皮肤文件夹内的对应文件和 skin.ini，如果有修改请注意备份！
  The following instructions will OVERWRITE corresponding files and skin.ini.
  Please back them up if there's any manual modification.

  1. 进入任何一个皮肤文件夹
     Enter any one skin folder
  2. 双击 APPLY.bat
     Double-click APPLY.bat
  3. 在 osu! 里重新加载皮肤：<Ctrl+Shift+Alt+s>
     Reload the skin in osu!: <Ctrl+Shift+Alt+s>
